# Python 学习路线

> 面向有 JavaScript / TypeScript 基础的开发者，用 30 天建立从 Python 基础语法到 AI Agent 工程化的完整路径。

## 总览

| 模块 | 范围 | 核心主题 | 建议入口 |
| --- | --- | --- | --- |
| Part 1-2 | Day 1 - 12 | 核心语法、数据结构、函数、OOP、异常、文件、模块 | [入门与核心语法](./phase-01-basics/) |
| Part 3-4 | Day 13 - 23 | 异步、HTTP、FastAPI、实时通信、正则、NumPy、Pandas、可视化 | [异步、数据与自动化](./phase-03-async/) |
| Part 5-6 | Day 24 - 30 | LLM API、Prompt、函数调用、RAG、Agent、测试、日志、Docker | [AI Agent 与工程化](./phase-05-agent/) |

## 学习顺序

1. 先完成 Phase 1，建立 Python 基础语法和运行时直觉。
2. 再完成 Phase 2，把代码组织、异常处理、文件读写和模块化补齐。
3. 如果你的目标是后端接口或 AI 服务接入，继续学习 Phase 3。
4. 如果你的目标包含数据处理、报表、自动化办公，再补 Phase 4。
5. 最后用 Phase 5 和 Phase 6 把能力串成可运行、可维护、可部署的 AI Agent 项目。
